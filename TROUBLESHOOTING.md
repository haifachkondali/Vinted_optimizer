# Troubleshooting Guide - Vinted Optimizer\n\n## Common Issues and Solutions\n\n### Installation & Setup\n\n#### Problem: \"ModuleNotFoundError: No module named 'vinted_scraper'\"\n\n**Solution**:\n```bash\npip install vinted_scraper==2.4.0\n```\n\nOr reinstall all dependencies:\n```bash\npip install -r requirements.txt\n```\n\n#### Problem: \"Python is not installed\" or \"command not found: python\"\n\n**Solution**:\n- Verify Python installation: `python --version`\n- If not found, download from https://www.python.org/downloads/\n- On Linux/Mac, use `python3` instead of `python`\n- Add Python to PATH if on Windows\n\n#### Problem: Virtual environment not activating\n\n**Solution**:\n```bash\n# Create new venv\npython -m venv .venv\n\n# Activate (choose correct one for your OS)\nsource .venv/bin/activate           # Linux/Mac/WSL\n.venv\\Scripts\\activate.bat          # Windows CMD\n.venv\\Scripts\\Activate.ps1          # Windows PowerShell\n```\n\n---\n\n### Scraping Issues\n\n#### Problem: \"No items found\" when using --user scraping\n\n**Causes & Solutions**:\n\n1. **Username typo** - Double check the Vinted username (case-sensitive)\n   ```bash\n   python main.py --user correct_username\n   ```\n\n2. **User has no public listings** - Some profiles hide listings\n   - Check if profile is public on Vinted website\n   - Try another username\n\n3. **Vinted blocking requests** - Too many requests too fast\n   ```bash\n   # Increase delay between requests\n   python main.py --user username --delay 5.0\n   ```\n\n4. **Network timeout** - Connection issues\n   ```bash\n   # Check internet connection\n   # Try again with increased delay\n   ```\n\n#### Problem: \"Scraping timeout\" or \"Connection refused\"\n\n**Solution**:\n```bash\n# Increase request delay\npython main.py --user username --delay 3.0 --max 10\n\n# Check your internet connection\n# Try from different network if possible\n```\n\n#### Problem: \"403 Forbidden\" or \"429 Too Many Requests\"\n\n**Solution**:\nVinted is rate-limiting or blocking your requests.\n\n1. Increase delay significantly:\n   ```bash\n   python main.py --user username --delay 5.0\n   ```\n\n2. Use fewer items:\n   ```bash\n   python main.py --user username --max 5 --delay 5.0\n   ```\n\n3. Wait 1-2 hours before retrying\n\n4. Switch to --file mode instead:\n   ```bash\n   # Create JSON file manually and use:\n   python main.py --file items.json\n   ```\n\n---\n\n### File Input Issues\n\n#### Problem: \"JSON decode error\" or \"Invalid JSON\"\n\n**Solution**:\n1. Validate JSON format: https://jsonlint.com/\n2. Common mistakes:\n   - Missing commas between items\n   - Trailing commas in arrays\n   - Single quotes instead of double quotes\n   - Unescaped special characters\n\n3. Required fields:\n   ```json\n   {\n     \"titre\": \"...\",        // Required\n     \"description\": \"...\"   // Required\n   }\n   ```\n\n#### Problem: \"File not found\"\n\n**Solution**:\n```bash\n# Use absolute path\npython main.py --file /full/path/to/items.json\n\n# Or relative path from current directory\npython main.py --file ./data/items.json\n```\n\n#### Problem: \"UnicodeDecodeError\"\n\n**Solution**:\n1. Save JSON file as UTF-8 without BOM\n2. In VS Code: Set encoding to \"UTF-8\"\n3. In Python:\n   ```python\n   with open('file.json', encoding='utf-8') as f:\n       data = json.load(f)\n   ```\n\n---\n\n### GDPR Export Issues\n\n#### Problem: \"No HTML files found\"\n\n**Solution**:\n1. Verify ZIP/folder contains Vinted data\n2. Extract ZIP manually and check structure:\n   - Should have folders like: `items/`, `messages/`, `listings/`\n   - Look for `index.html` files\n\n3. Try explicit folder:\n   ```bash\n   python main.py --export-folder ./vinted_data/\n   ```\n\n#### Problem: \"Unable to extract ZIP file\"\n\n**Solution**:\n```bash\n# Extract manually first\nunzip vinted_export.zip -d vinted_data/\n\n# Then use folder mode\npython main.py --export-folder ./vinted_data/\n```\n\n#### Problem: \"No articles found in export\"\n\n**Solution**:\n1. Check export contains listings (not just messages/feedback)\n2. Try different HTML files in the folder\n3. Check file encoding\n\n---\n\n### Analysis & Output Issues\n\n#### Problem: \"Permission denied\" when saving output\n\n**Solution**:\n```bash\n# Ensure output directory is writable\n# Use different output path\npython main.py --file items.json --output ./my_reports/\n\n# On Windows\npython main.py --file items.json --output C:\\Reports\\vinted\n```\n\n#### Problem: \"Output folder already exists with files\"\n\n**Solution**:\n```bash\n# Use unique output folder\npython main.py --file items.json --output ./reports_2024_05_15/\n\n# Or clean old folder first\nrm -rf output_vinted/\npython main.py --file items.json\n```\n\n#### Problem: \"Memory error\" or \"Slow processing\"\n\n**Solution**:\n```bash\n# Process fewer items at once\npython main.py --user username --max 5\n\n# Or split into multiple runs with different files\n```\n\n---\n\n### Test Issues\n\n#### Problem: \"pytest: command not found\"\n\n**Solution**:\n```bash\npip install pytest\npytest tests/ -v\n```\n\n#### Problem: \"Test failures\"\n\n**Solution**:\n```bash\n# Run with verbose output\npytest tests/ -v --tb=long\n\n# Check if all dependencies installed\npip install -r requirements.txt\n```\n\n---\n\n### Performance Issues\n\n#### Problem: \"Script takes too long\"\n\n**Solution**:\n1. Reduce number of items:\n   ```bash\n   python main.py --user username --max 10\n   ```\n\n2. Reduce delay:\n   ```bash\n   python main.py --user username --delay 0.5\n   ```\n\n3. Use --file mode instead of scraping (much faster)\n\n---\n\n### Encoding Issues\n\n#### Problem: \"Special characters not displaying correctly\"\n\n**Solution**:\n1. Ensure terminal supports UTF-8:\n   ```bash\n   # Set environment variable (Linux/Mac)\n   export LANG=en_US.UTF-8\n   ```\n\n2. Check output files are UTF-8:\n   - VS Code: Should auto-detect\n   - Windows: Use Notepad++ and set to UTF-8\n\n---\n\n## Getting Help\n\nIf issues persist:\n\n1. **Check logs**: Run with --debug flag (if supported)\n2. **Verify setup**: Run `pytest tests/ -v`\n3. **Check versions**:\n   ```bash\n   python --version\n   pip show vinted_scraper rich typer\n   ```\n4. **Search known issues**: Check GitHub issues\n5. **Report bug**: Create detailed bug report with:\n   - Python version\n   - OS and version\n   - Command used\n   - Full error message\n   - Steps to reproduce\n
